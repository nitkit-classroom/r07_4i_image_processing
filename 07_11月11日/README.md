# 1. 画像処理の基礎に関する次の問いに答えよ
1. コンピュータビジョンとコンピュータグラフィックスの違いについて説明せよ
2. （狭義の）画像処理と画像解析の違いについて説明せよ
3. 図1にピンホールカメラで被写体を撮影した様子を示す．「光軸」「ピンホール」「焦点距離」「投影面」がそれぞれどれにあたるか．図に加筆しながらどれがどれか矢印で示せ
4. ピンホールカメラで表されるものは透視投影モデルである．この文章が正しいか否か答えよ
5. 図2はどちらも同じ場所から別のカメラで撮影した猫の画像である．画角が広いのは(a)(b)のどちらか答えよ
6. 図3について，[a]では壁の模様（縦線）が正しく読み取れるが，[b]ではU字状の模様が現れている．何故このようなことが起きるか．「標本化」「標本化間隔」という言葉を用いて説明せよ（必要に応じて教科書P.45図2.35を参照すること・試験本番では教科書を参照させる問題の出し方はしない）
7. 図4について(a)〜(d)の全てがグレースケール画像であるにも関わらず，特に(c)(d)においては顕著に「輪郭」が現れている
	1. この輪郭のことを何と呼ぶか
	2. この輪郭は何故起きるか
	3. (c)と(d)では，(c)のほうが輪郭の数が多い．何故か
8. 混色について答えよ
	1. プリンタでは3色（場合によってはそれ以上）のインクを用紙に重ね合わせて色を生み出している．この混色手法の名前を答えよ
	2. カラーモニタでは，3色（場合によってはそれ以上）の発光体を全て光らせて白色，必要に応じて発光体を消灯することで色を生み出している．この混色手法の名前を答えよ
9. HD画質の画像データのデータ量を，単位をバイトで答えよ．ただしHD画質とは横1280px縦720pxの画像を指す
10. [R 0xFF，G 0x10，B 0x20]の値を持つ画素について，HSVモデル上での数値へ変換せよ．ただし値域は，0 <= S, I <= 1と0 <= H <= 2πとする
